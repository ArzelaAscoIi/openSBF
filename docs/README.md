# Data Sources

This folder contains the official reference materials that the openSBF app is based on:

- **Fragenkatalog-See-August-2023.pdf** — Official question catalogue for the SBF See (inland waterways) exam (August 2023 edition)
- **Fragenkatalog-Binnen-August-2023.pdf** — Official question catalogue for the SBF Binnen (inland waterways) exam (August 2023 edition)
- **sichtzeichen_sbf_see.pdf** — Visual signals reference for the SBF See exam

These documents are published by the German Federal Waterways and Shipping Administration (WSV) and serve as the authoritative source for all questions and learning content in the app.
